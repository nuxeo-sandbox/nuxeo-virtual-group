# nuxeo-virtual-group


## List of Features (details below)

This plugin provides an example of Virtual Group configuration with Nuxeo.

## Build

Assuming maven is correctly setup on your computer:

```
git clone 
mvn clean install
```

## Install

Install the package on your instance.
